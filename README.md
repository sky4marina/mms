# Github issue client

Test application to get issue from Facebook repository for react. This application was bootstrapped with `create-react-app`

## Getting Started

1. `yarn install` to add all dependencies
2. `yarn start` to run app locally

## Tests
1. `yarn test a` run all tests
2. `yarn test a --coverage` run tests with coverage

## Build
1. `yarn build` builds prod version
2. `serve -s build` to run build locally
